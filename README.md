Daily Planner Web Application

 Overview

This project is a full-stack daily planner web application that helps users manage their day effectively by organizing tasks, tracking small achievements, and scheduling events.

---

## Features

* User login system
*  To-do list for daily tasks
*  Small wins tracker
*  Events management
*  Profile page
*  Memories page
*  Calendar section

---

## Tech Stack

* **Frontend:** React
* **Backend:** Flask (Python)
* **Database:** SQLite

---

##  How to Run Locally

###  Backend

cd backend
pip install -r requirements.txt
python app.py

###  Frontend

cd frontend/dayplanner
npm install
npm start

---

##  Deployment

* Frontend deployed using Vercel
* Backend can be run locally (Flask server)
